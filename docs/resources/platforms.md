# 在线评测平台

<DifficultyBadge level="easy" />

## 国内平台

### 1. 洛谷

- **网址**：https://www.luogu.com.cn/
- **特点**：
  - 中文界面，适合初学者
  - 题目分类清晰，有难度标签
  - 题解丰富，社区活跃
  - 支持团队功能
  - 有在线 IDE
- **推荐指数**：⭐⭐⭐⭐⭐
- **适合人群**：所有水平

### 2. 牛客网

- **网址**：https://www.nowcoder.com/
- **特点**：
  - 题库丰富
  - 有竞赛模拟
  - 适合求职准备
  - 有专项练习
- **推荐指数**：⭐⭐⭐⭐
- **适合人群**：初学者、求职者

### 3. 计蒜客

- **网址**：https://www.jisuanke.com/
- **特点**：
  - 有系统的课程
  - 适合初学者入门
  - 有在线编程环境
- **推荐指数**：⭐⭐⭐⭐
- **适合人群**：初学者

### 4. 蓝桥云课

- **网址**：https://www.lanqiao.cn/
- **特点**：
  - 蓝桥杯官方平台
  - 有历年真题
  - 适合备赛
- **推荐指数**：⭐⭐⭐⭐
- **适合人群**：蓝桥杯参赛者

## 国际平台

### 1. Codeforces

- **网址**：https://codeforces.com/
- **特点**：
  - 国际知名竞赛平台
  - 定期举办比赛
  - 有 Rating 系统
  - 题目质量高
  - 题解详细
- **推荐指数**：⭐⭐⭐⭐⭐
- **适合人群**：有一定基础，想提高

### 2. AtCoder

- **网址**：https://atcoder.jp/
- **特点**：
  - 日本平台
  - 题目质量高
  - Beginner Contest 适合入门
  - Regular Contest 难度较高
- **推荐指数**：⭐⭐⭐⭐⭐
- **适合人群**：初学者到高手

### 3. LeetCode

- **网址**：https://leetcode.cn/
- **特点**：
  - 面向求职
  - 中文版友好
  - 题目偏算法应用
  - 有企业题库
- **推荐指数**：⭐⭐⭐⭐⭐
- **适合人群**：求职者、面试准备

### 4. USACO

- **网址**：http://www.usaco.org/
- **特点**：
  - 美国信息学竞赛
  - 有训练题库
  - 月赛制度
- **推荐指数**：⭐⭐⭐⭐
- **适合人群**：想参加国际竞赛

## 使用技巧

### 1. 洛谷题单

```
洛谷有官方题单：
- 入门题单
- 普及题单
- 提高题单
- 省选题单

按题单刷题，循序渐进
```

### 2. Codeforces 比赛

```
- Div. 4：适合初学者
- Div. 3：入门后
- Div. 2：有一定基础
- Div. 1：高手

建议从 Div. 4 开始参加
```

### 3. AtCoder 比赛

```
- ABC（Beginner Contest）：适合初学者
- ARC（Regular Contest）：提高
- AGC（Grand Contest）：高手

ABC 的 A、B 题通常很简单
```

## 刷题建议

### 1. 循序渐进

```
1. 先完成入门题（A+B Problem 等）
2. 学习基础语法后，刷语法题
3. 学习算法后，刷对应算法的题目
4. 参加模拟赛，检验水平
```

### 2. 分类刷题

```
按算法分类：
- 先刷完所有排序题
- 再刷完所有搜索题
- 然后刷 DP 题
- ...
```

### 3. 定期比赛

```
- 每周参加 1-2 场 Codeforces/AtCoder
- 记录 Rating 变化
- 赛后补题
```

### 4. 题解学习

```
- 不会的题目看题解
- 理解后自己写一遍
- 总结套路和模板
```

## 总结

- **洛谷**：最适合中国学生的入门平台
- **Codeforces**：提高水平的首选
- **AtCoder**：题目质量高，适合系统学习
- **LeetCode**：求职面试必备
- 建议多平台结合使用
- 坚持刷题，定期比赛
