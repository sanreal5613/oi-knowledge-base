# 运算符与表达式

## 算术运算符

| 运算符 | 说明 | 示例 | 结果 |
|--------|------|------|------|
| `+` | 加法 | `5 + 3` | 8 |
| `-` | 减法 | `5 - 3` | 2 |
| `*` | 乘法 | `5 * 3` | 15 |
| `/` | 除法 | `5 / 2` | 2（整除） |
| `%` | 取模 | `5 % 2` | 1 |
| `++` | 自增 | `a++` | a = a + 1 |
| `--` | 自减 | `a--` | a = a - 1 |

### 整数除法

```cpp
int a = 7, b = 2;
cout << a / b << endl;     // 3（整除，舍去小数）
cout << a % b << endl;     // 1（余数）

// 要得到浮点结果，需要类型转换
double result = (double)a / b;
cout << result << endl;    // 3.5
```

### 自增自减

```cpp
int a = 5;
cout << a++ << endl;  // 输出 5，然后 a 变成 6（后置）
cout << ++a << endl;  // a 先变成 7，然后输出 7（前置）

// 竞赛中推荐使用前置 ++i，效率略高
for (int i = 0; i < n; ++i) {
    // ...
}
```

## 关系运算符

| 运算符 | 说明 | 示例 | 结果 |
|--------|------|------|------|
| `==` | 等于 | `5 == 5` | true |
| `!=` | 不等于 | `5 != 3` | true |
| `>` | 大于 | `5 > 3` | true |
| `<` | 小于 | `5 < 3` | false |
| `>=` | 大于等于 | `5 >= 5` | true |
| `<=` | 小于等于 | `5 <= 3` | false |

```cpp
int a = 10, b = 20;
if (a < b) {
    cout << "a 小于 b" << endl;
}
```

::: warning 注意
判断相等用 `==`，不是 `=`！`=` 是赋值运算符。
:::

## 逻辑运算符

| 运算符 | 说明 | 示例 | 结果 |
|--------|------|------|------|
| `&&` | 逻辑与（AND） | `true && false` | false |
| `\|\|` | 逻辑或（OR） | `true \|\| false` | true |
| `!` | 逻辑非（NOT） | `!true` | false |

### 真值表

```cpp
// && (与)：全真才真
true && true   // true
true && false  // false
false && false // false

// || (或)：有真即真
true || false  // true
false || false // false

// ! (非)：取反
!true   // false
!false  // true
```

### 短路求值

```cpp
int a = 0, b = 5;

// && 短路：前面为 false，后面不执行
if (a != 0 && b / a > 2) {  // 不会除以 0，因为短路
    // ...
}

// || 短路：前面为 true，后面不执行
if (a == 0 || b / a > 2) {  // 前面已经 true，后面不执行
    // ...
}
```

## 位运算符

| 运算符 | 说明 | 示例 | 结果 |
|--------|------|------|------|
| `&` | 按位与 | `5 & 3` | 1 |
| `\|` | 按位或 | `5 \| 3` | 7 |
| `^` | 按位异或 | `5 ^ 3` | 6 |
| `~` | 按位取反 | `~5` | -6 |
| `<<` | 左移 | `5 << 1` | 10 |
| `>>` | 右移 | `5 >> 1` | 2 |

### 位运算详解

```cpp
// 二进制表示
// 5 = 0101
// 3 = 0011

5 & 3   // 0001 = 1（都是 1 才是 1）
5 | 3   // 0111 = 7（有 1 就是 1）
5 ^ 3   // 0110 = 6（不同为 1）

// 左移：乘以 2
5 << 1  // 10（5 × 2）
5 << 2  // 20（5 × 4）

// 右移：除以 2
5 >> 1  // 2（5 ÷ 2）
5 >> 2  // 1（5 ÷ 4）
```

### 位运算技巧

```cpp
// 判断奇偶
if (n & 1) {
    cout << "奇数" << endl;
} else {
    cout << "偶数" << endl;
}

// 快速乘除 2 的幂
int a = n << 3;  // n × 8
int b = n >> 2;  // n ÷ 4

// 交换两个数（不用临时变量）
a ^= b;
b ^= a;
a ^= b;

// 获取第 k 位（从右往左，从 0 开始）
bool bit = (n >> k) & 1;

// 设置第 k 位为 1
n |= (1 << k);

// 清除第 k 位（设为 0）
n &= ~(1 << k);

// 翻转第 k 位
n ^= (1 << k);
```

## 赋值运算符

| 运算符 | 说明 | 等价于 |
|--------|------|--------|
| `=` | 赋值 | - |
| `+=` | 加后赋值 | `a = a + b` |
| `-=` | 减后赋值 | `a = a - b` |
| `*=` | 乘后赋值 | `a = a * b` |
| `/=` | 除后赋值 | `a = a / b` |
| `%=` | 取模后赋值 | `a = a % b` |

```cpp
int a = 10;
a += 5;   // a = 15
a *= 2;   // a = 30
a %= 7;   // a = 2
```

## 运算符优先级

从高到低：

1. `()` 括号
2. `!` `~` `++` `--` 单目运算符
3. `*` `/` `%` 乘除模
4. `+` `-` 加减
5. `<<` `>>` 移位
6. `<` `<=` `>` `>=` 关系
7. `==` `!=` 相等
8. `&` 按位与
9. `^` 按位异或
10. `|` 按位或
11. `&&` 逻辑与
12. `||` 逻辑或
13. `=` `+=` `-=` 等赋值

::: tip 建议
不确定优先级时，**多用括号**！括号不影响效率，但能避免错误。
:::

## 示例代码

```cpp
#include <iostream>
using namespace std;

int main() {
    int a = 10, b = 3;
    
    cout << "算术运算:" << endl;
    cout << "a + b = " << (a + b) << endl;
    cout << "a - b = " << (a - b) << endl;
    cout << "a * b = " << (a * b) << endl;
    cout << "a / b = " << (a / b) << endl;
    cout << "a % b = " << (a % b) << endl;
    
    cout << "\n位运算:" << endl;
    cout << "a & b = " << (a & b) << endl;
    cout << "a | b = " << (a | b) << endl;
    cout << "a ^ b = " << (a ^ b) << endl;
    cout << "a << 1 = " << (a << 1) << endl;
    cout << "a >> 1 = " << (a >> 1) << endl;
    
    return 0;
}
```

2. 判断一个数 n 是否是 2 的幂（提示：用位运算）
3. 不用临时变量交换两个整数

<details>
<summary>练习 2 答案</summary>

```cpp
// 2 的幂的二进制只有一个 1
// 例如：8 = 1000, 16 = 10000
// n & (n-1) 会清除最右边的 1
// 如果 n 是 2 的幂，结果为 0

bool isPowerOfTwo(int n) {
    return n > 0 && (n & (n - 1)) == 0;
}
```

## 总结

- 算术运算符：`+` `-` `*` `/` `%`
- 关系运算符：`==` `!=` `>` `<` `>=` `<=`
- 逻辑运算符：`&&` `||` `!`
- 位运算符：`&` `|` `^` `~` `<<` `>>`
- 位运算在竞赛中非常有用，要熟练掌握！
- 不确定优先级时多用括号
